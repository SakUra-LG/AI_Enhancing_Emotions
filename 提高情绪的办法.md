### 提高情绪的办法（最小可运行指引）

本指南汇总“情绪增强”所需的最小文件清单、各文件作用与用法，便于快速在另一台机器上复现并运行。适用于当前项目目录：`bert_excitation_train/AI_Novle`

---

## 一、最小文件清单

- 核心脚本（必须）
  - `bert_excitation_train/scripts/generate_chapter_content.py`
  - `bert_excitation_train/scripts/emotion_analyzer.py`
  - `bert_excitation_train/scripts/optimized_rule_scorer.py`
  - `bert_excitation_train/scripts/smart_sample_search.py`
  - 参考文档（可读以便理解）：`bert_excitation_train/scripts/EMOTION_MAXIMIZATION_TECHNIQUES.md`

- 数据（RAG 样本库，必须）
  - `bert_excitation_train/data/universal_samples_vectors.npy`
  - `bert_excitation_train/data/universal_samples_data.json`
  - 可选但推荐：`bert_excitation_train/data/universal_samples.txt`
  - 可选但推荐：`bert_excitation_train/data/universal_samples_scored.txt`

- 运行上下文（提供可直接使用的样例）
  - `bert_excitation_train/outputs/master_ctx_v2.txt`
  - `bert_excitation_train/outputs/prev_life_ctx_v2.txt`
  - 目录：`bert_excitation_train/outputs/chapters/`（可为空，程序会写入章节结果）

- 依赖清单
  - `bert_excitation_train/requirements.txt`（如不存在，建议新建并写入下方依赖）

---

## 二、各文件的作用

- `generate_chapter_content.py`：章节生成主脚本。内置情绪硬性约束提示、情绪评分与规则评分、迭代式反馈循环（未达标自动再生），并进行多版本择优。
- `emotion_analyzer.py`：情绪分析器。综合“分类置信度、情绪词密度、情绪深度、情绪转折强度”等维度，输出0-1情绪强度分数。
- `optimized_rule_scorer.py`：规则评分器。对生成文本进行长度/结构/要素等规则打分，与情绪强度共同决定是否达标。
- `smart_sample_search.py`：相似样本检索（RAG）。利用中文句向量模型（如 bge_large_zh）基于向量库检索最相关示例，增强生成质量与情绪。
- `EMOTION_MAXIMIZATION_TECHNIQUES.md`：方法论总览。总结项目中用于“情绪最大化”的关键技术点，便于快速理解和调参。
- `universal_samples_vectors.npy`、`universal_samples_data.json`：相似样本向量库与元数据（必须）。若缺失，RAG 检索将无法进行。
- `master_ctx_v2.txt`、`prev_life_ctx_v2.txt`：示例上下文（世界观/设定与“上一世”信息）供直接生成使用。

## 四、必要环境与本地化改动

- API Key（通义千问/阿里云 DashScope）
  - 在 `bert_excitation_train/scripts/generate_chapter_content.py` 中替换为自己的 Key，或改为读取环境变量（推荐）。

- 中文句向量模型路径（bge_large_zh）
  - 在 `bert_excitation_train/scripts/smart_sample_search.py` 顶部有 `model_path` 配置。请改为目标机器上实际的 bge_large_zh 路径，或直接改为在线模型名（如 `BAAI/bge-large-zh`）以自动下载。

- RAG 样本库
  - 确保目标机器存在 `data/universal_samples_vectors.npy` 与 `data/universal_samples_data.json` 两个文件。若需要重新构建，可参考 `scripts/handle_universal_samples.py`（可选）。

## 五、最常用的运行方式

1) 直接批量生成章节（使用现有样例上下文）

```bash
python bert_excitation_train/scripts/generate_chapter_content.py ^
  --chapter 6 ^
  --batch 5 ^
  --versions 1 ^
  --iterations 2 ^
  --min-emotion 0.6
```

2) 指定上下文文件生成

```bash
python bert_excitation_train/scripts/generate_chapter_content.py ^
  --chapter 6 ^
  --batch 3 ^
  --master-ctx bert_excitation_train/outputs/master_ctx_v2.txt ^
  --prev-life-ctx bert_excitation_train/outputs/prev_life_ctx_v2.txt
```

3) 运行“情绪反馈循环”演示（可选）

```bash
python bert_excitation_train/scripts/test_emotion_feedback_loop.py
```

## 六、调参关键点（情绪相关）

- `--min-emotion`：最低情绪强度阈值（0-1）。建议从 `0.6` 起步。
- `--iterations`：单次生成的最大反馈循环次数。阈值未达标自动再生，推荐 `2-3`。
- 提示词硬约束：`generate_chapter_content.py` 中已强制“多维情绪/深度/转折/密度”等要求，可在同文件中微调中文提示以适配题材。
- 规则评分：`optimized_rule_scorer.py` 可调节长度/结构权重，配合情绪强度共同筛选。
- RAG：`smart_sample_search.py` 的检索数量与相似度阈值会影响风格与情绪共鸣度，可按需要微调。

---

## 七、常见问题

- 向量库缺失或维度不匹配
  - 请确认 `universal_samples_vectors.npy` 与 `universal_samples_data.json` 配套；若自建，需使用相同模型、相同维度的向量。

- 模型路径错误或加载失败
  - 将 `smart_sample_search.py` 的 `model_path` 改为实际可访问的本地目录，或使用在线模型名以自动下载。

- Key 配置无效
  - 将 Key 设置为环境变量或直接替换脚本中的占位值；若使用环境变量，请同步修改脚本读取逻辑。
